# AutoTranslateMod

若不显示图片需科学上网

这是一个自动翻译Localizer Mod导出的原始文本的程序，同时能够支持跳过原有中文翻译。
首先，先到http://api.fanyi.baidu.com/ 注册百度开发者账号（需要高级版,免费）得到Appid和密钥。

------------
